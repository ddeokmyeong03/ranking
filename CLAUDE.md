# 수정사항
1. 근무표가 결국 메인 서비스입니다. 홈과 근무표를 구분하지 않고, '안녕하세요'가 있는 칸, 근무표, 다가오는 당직 ... 순으로 위에서 아래로 정렬된 메인 페이지였으면 좋겠습니다. 
2. 1번과 같이 수정하면, 메뉴가 홈, 변경희망, 요청, 수당, 프로필이 있습니다. 이 좌측 네비게이션 바를 하단 네비게이션 바로 변경하고 싶습니다. '변경 희망, 요청, 홈(색깔 다르게 해서 홈 버튼 강조), 수당, 프로필' 순으로 되있으면 좋겠습니다.

# 보충사항
1. 타 사용자가 게시한 변경 희망 게시글에 변경 신청을 하면, 신청을 한 사람이 언제랑 변경하기를 신청했는지 알 수 있었으면 좋겠음.

# 질문 사항
1. 변경 요청 게시글이나 이런 요청 게시글이 초번에 사라졌다 다시 나타났다 하는데, 이것은 그냥 로컬에서 테스트해서 버벅거리는 거겠죠?
