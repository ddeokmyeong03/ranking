# 2차 피드백 (관리자페이지)
1. (좌측 네비게이션바) 모바일 버전에서는 좌측 네비게이션 바가 보이지 않습니다. 사용자 페이지처럼 하단 고정 네비게이션 바가 좋을 것 같습니다. 아니면, 좌측 네비게이션 바를 없애고, 대시보드에 다 있었으면 좋겠습니다. 그러려면 간부관리, 사이클 관리같은 페이지 상단 좌측에 '뒤로가기' 버튼이 있어야 될 것 같습니다.
2. 사이클 생성(대시보드에 있는 버튼)과 사이클 관리(좌측 네비게이션 바)를 '사이클 관리'로 통일했으면 좋겠습니다. 사이클 관리 페이지로 들어가서 생성을 하면 되기 때문입니다.
3. 모바일에서 볼때는 간부관리 페이지에 있는 표가 기존 사용자 화면의 근무표처럼 정렬되어 있지 않게 보입니다. 이거도 해결하고 싶습니다.
4. 대시보드 최하단에 사용자 페이지로 이동 버튼도 있었으면 좋겠습니다.
5. 활성 사이클이 반영되지 않는 것 같습니다.
6. 사이클 관리에서 현재 반영된 사이클이 무엇인지 표현하고 싶습니다. 카드 우측 하단에 '현재 편성중'이라는 단어와 초록색 점이 있으면 좋을 것 같습니다.
