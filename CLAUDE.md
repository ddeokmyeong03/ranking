# 육군 간부 당직근무표 변경 시스템

1. 아이템 소개 : 이 서비스는 육군 간부 전용으로 당직근무를 쉽게 변경하고, 변경 즉시 최신화되어 당직근무표가 자동으로 설정되는 간편한 서비스입니다.
2. 필요 서비스
 - 관리자용 서비스 : 부대코드 등록, 당직근무 사이클 지정, 사용자 등록 등의 권한을 가진 관리자용 서비스
 - 사용자용 서비스 : 당직근무 일정 확인, 당직근무 변경 희망 등록, 당직근무 변경 신청 등록, 당월 당직근무 수당 자동 계산 등을 지닌 서비스
3. 기능
 ## 회원 등록 및 로그인 기능
 - 회원가입 / 로그인 : 아이디는 본인의 군번, 비밀번호는 직접 설정하도록 하여 간부만 사용할 수 있도록 한다.
 - 2차 로그인 (부대코드) : 관리자가 발급받은 본인 사단 및 여단의 부대코드를 입력하여 2차 로그인을 할 수 있도록 한다.
 - 자동 로그인 : 신뢰하는 기기에 대해 자동으로 로그인을 해주는 기능
 - 회원정보 수정 : 본인의 정보 (보직, 임관일, 전역일, 비밀번호 등)를 수정할 수 있도록 하는 기능
 ## 메인 기능
 - 당직근무 자동 편성 기능 : 관리자가 사이클을 등록하면, 자동으로 평일 당직, 주말 주간 당직, 주말 야간 당직 등을 자동으로 편성해주는 기능 (자체 AI)
 - 공휴일, 전투휴무 등록 기능 (관리자용) : 평일에 사단 및 여단에 전투휴무가 생기면 전투휴무에도 주말 주간당직, 주말 야간당직으로 변환됨. 이를 관리자가 전투휴무 일자를 등록하면 자동으로 최신화됨

 ## 본인의 당직근무 변경을 직접 등록하는 경우의 기능
 - 당직근무 변경 등록 기능 : 사용자는 본인의 근무가 있는 날에 변경을 희망하는 경우, 모든 사용자에게 그 날 당직근무 바꿔주실 분을 알리기 위해 등록하고, 타 사용자에게 알림이 전달되는 기능. 달력의 해당 일자의 칸에 색상을 변환하여 이 날에 당직근무 변경을 희망하는 사람이 있다는 것을 알릴 수 있음
 - 당직근무 변경 신청 기능 : 사용자가 게시한 당직근무 변경 등록을 한 내역에 승인을 누르면 본인의 근무 중 희망하는 날짜를 입력하고 신청하여 당직근무 변경 등록을 한 사용자에게 '저는 00월 00일 근무인데, 변경 가능한가요?'라는 알람이 자동으로 오게 하는 기능
 - 당직근무 변경 신청 승인 기능 : 변경 가능하냐는 알림에 승인을 하면 자동으로 변경되며, 거절하면 '죄송하지만 제가 그때 일정이 있어 변경이 어려울 것 같습니다'와 같은 알람을 보내 정중히 거절하는 기능

 ## 희망하는 날짜로 당직근무 변경을 요청하는 기능
 - 당직근무 변경 요청 기능 : 사용자는 해당하는 날짜에 당직근무를 서고 싶어하는 경우, 해당 날짜에 당직근무 변경을 요청하여, 해당 일에 편성된 사용자에게 알암을 가게 하는 기능 '제가 00일에 ~~ 당직근무인데, 이 날짜와 변경을 희망합니다.'
 - 당직근무 변경 요청 승인 기능 : 알람을 받은 사용자는 해당 날짜 변경을 승인하면, 자동으로 변경되며, 거절하면 '죄송하지만 제가 그때 일정이 있어 변경이 어려울 것 같습니다'와 같은 알람을 보내 정중히 거절하는 기능
